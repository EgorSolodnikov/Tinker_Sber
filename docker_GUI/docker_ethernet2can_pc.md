# Docker окружение для сборки проекта (на pc)

## Подготовка образа `gui_and_joy`

```bash
sudo rm -rf build/ log/ install/
cd docker_GUI
./build_img.sh
```

## Запуск контейнера и сборка

```bash
./build_cont.sh
```

Скрипт смонтирует корень репозитория в `/workspace`, выполнит `colcon build --symlink-install` и оставит интерактивную консоль внутри контейнера. Все артефакты (`build/`, `install/`, `log/`) остаются на хосте.

## Чтобы открыть дополнительные терминалы

```bash
./new_window_cont.sh
```

Откроется новая сессия в уже запущенном контейнере с автоматически загруженным окружением ROS2.
